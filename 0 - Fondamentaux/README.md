# Module 0 — Fondamentaux

Socle mathématique et informatique couvrant le programme prépa MP2I/MPI (2 ans). Fondation requise pour les modules 1-8.

## Prérequis

- Niveau bac scientifique (terminale spé maths + NSI recommandé)

## Objectifs

- Maîtriser l'arithmétique modulaire et les structures algébriques (fondations crypto)
- Analyser la complexité d'un algorithme et choisir la bonne structure de données
- Implémenter les algorithmes fondamentaux (tri, graphes, prog dynamique)
- Comprendre les automates, langages formels et la théorie de la complexité
- Manipuler la mémoire bas niveau (C) et les concepts de concurrence

## Programme de référence

Ce module couvre le programme officiel des classes préparatoires MP2I (1ère année) et MPI (2ème année), filière d'accès aux concours Centrale-Supélec, Mines-Ponts et CCINP.
