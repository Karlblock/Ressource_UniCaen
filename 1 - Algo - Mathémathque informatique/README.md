# Module 1 — Algorithmes & Mathématiques informatiques

Fondations algorithmiques et mathématiques nécessaires pour la crypto, l'IA et le pentest.

## Prérequis

- Module 0 (CS50) — bases de la programmation

## Sommaire

### Cours

| Fichier | Sujet |
| --- | --- |
| `cours/complexite.md` | Notation Big-O, analyse d'algorithmes |
| `cours/structures_donnees.md` | Listes, piles, files, arbres, graphes |
| `cours/tri_recherche.md` | Algorithmes de tri, recherche binaire |
| `cours/graphes.md` | BFS, DFS, Dijkstra, applications réseau |
| `cours/logique_booleenne.md` | Portes logiques, algèbre de Boole |

### Labs

| Fichier | Exercice |
| --- | --- |
| `labs/exercices_tri.py` | Implémentation des tris classiques |
| `labs/exercices_graphes.py` | Parcours de graphes, plus court chemin |

## Objectifs

- Analyser la complexité d'un algorithme
- Choisir la structure de données adaptée à un problème
- Implémenter les algorithmes fondamentaux de tri et de recherche
- Modéliser un réseau comme un graphe et le parcourir
